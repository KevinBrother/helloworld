# 各种项目的hello world

## 采用分支的形式管理

- [状态]分支
  - 项目名
  - 便签、描述
  - 主要做了什么
  - 文档链接

## 新建文件夹下新增的文档需要只要要包含两个主题：作用、目的。如

```
# 文档

## 作用

## 目的

## 参考链接
```

## master 分支主要作用是编写统一文档与全局控制作用
