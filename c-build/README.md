# 操作手册

## 基本操作

1. 编写 CMakeLists.txt 文件

2. 调用 cmake 会根据 CMakeLists.txt 生成 Makefile

``` bash
    cd cmake && cmake .
```

3. 调用 make 会根据 Makefile 编译生成可执行文件

``` bash
make
```

4. 运行可执行文件

``` bash
./bin/helloCMake
```
